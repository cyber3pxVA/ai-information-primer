## How to Add New PDFs:

1. Place your PDF files in this directory
2. Update the website's HTML to include links to the new files
3. Ensure file names are descriptive and follow the naming convention

## File Naming Convention:

- Use lowercase letters
- Use hyphens to separate words
- Include descriptive keywords
- End with `.pdf` extension

Example: `transformer-architecture-explained.pdf`

## Notes:
